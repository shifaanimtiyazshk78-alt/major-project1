Frequently Asked Questions

How do I track my parcel?

Go to the Tracking page and enter your Tracking ID. With the backend running, status comes from the database (create matching orders in Admin).

How can I book a shipment?

Register for an account and use the Parcel Booking Form. Enter sender and recipient details, parcel specifications, and select your service type to get an instant quote.

What payment methods are accepted?

We accept online payments via integrated gateways. Walk-in customers can also pay by cash at branch locations.

How do I raise a support ticket?

Use the contact form below. Select the relevant category and provide your Tracking ID (if applicable). Our team will respond within 24–48 hours.

Frontend & backend

Run the Python app from backend/python_app so the site and API share one server (recommended): 

cd backend/python_app && uvicorn main:app --reload --host 127.0.0.1 --port 8000

Then open http://127.0.0.1:8000/ — tracking, registration, and login call the API automatically. Interactive API: /docs.

If you open HTML files directly (file://), set the API URL via the Admin dashboard toolbar (saved in the browser) or ensure the backend runs at http://127.0.0.1:8000.
